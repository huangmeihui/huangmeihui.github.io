Debian 安装 Oh-My-Zsh

2019-10-07 约 205 字 预计阅读 1 分钟

文章目录

  • 安装 Git 以及 zsh
1
2

sudo apt update
sudo apt install git zsh
  • 下载项目到本地
1

git clone https://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh --depth 1
  • 如果根目录之下有 .zshrc 这个 zsh 的配置文件则备份
1

cp ~/.zshrc ~/.zshrc.back
  • 通过 oh-my-zsh 的配置模板新建 zsh 的配置文件
1

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
  • 安装工作完毕，更换默认 shell ，需要重启会话
1

chsh -s /bin/zsh
  • 重新打开一个终端，可能会发现乱码，原因是没有安装 PowerLine 的字体（不同的 Linux 发行版可能有不同的包，建议先搜索一下）
1

sudo apt install fonts-powerline
  • Theme更换,编辑 .zshrc 文件,找到 ZSH_THEME 这一行修改即可 （内置主题列表）
1
2
3

vim ~/.zshrc
## 举个栗子
ZSH_THEME = "agnoster"
  • 手动更新
1

upgrade_oh_my_zsh
