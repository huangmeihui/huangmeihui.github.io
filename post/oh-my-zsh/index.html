Debian 安装 Oh-My-Zsh

2018-01-10 约 249 字 预计阅读 1 分钟

文章目录

  1. 安装Git以及zsh

    1

    # apt-get install git zsh

  1. 下载项目到本地

    1

    git clone https://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh

  2. 如果根目录之下有.zshrc这个zsh的配置文件则备份

    1

    cp ~/.zshrc ~/.zshrc.back

  3. 通过oh-my-zsh的配置模板新建zsh的配置文件

    1

    cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

  4. 安装工作完毕，更换默认shell，需要重启会话

    1

    chsh -s /bin/zsh

  5. 重新打开一个终端，可能会发现乱码，原因是没有安装PowerLine的字体（不同的Linux发行版可能有不同的包，建议先搜索一下）

    1

    # apt-get install fonts-powerline

  6. Theme更换,编辑.zshrc文件,找到ZSH_THEME这一行修改即可 （内置主题列表）

    1
2
3

    vim ~/.zshrc
## 举个栗子
ZSH_THEME = "agnoster"

  7. 手动更新

    1

    upgrade_oh_my_zsh
